# CLAUDE.md

## Project Overview
InvoiceKirim - Micro SaaS untuk bikin dan kirim invoice dalam hitungan detik.
Target user: Freelancer dan UMKM Indonesia.

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS
- TypeScript
- Supabase (akan ditambah di sesi berikutnya)

## Design Direction
- Aesthetic: Dark Elegant
- Vibe: Professional, friendly, trustworthy

## Color Palette
- Background: Black
- Text: White
- Accent: Red

## Typography
- Heading: Clash Display
- Body: DM Sans
- JANGAN pakai: Inter, Roboto, Arial

## Commands
- `npm run dev` - Start development server
- `npm run build` - Build untuk production
